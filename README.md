# Notion-Powered Next.js Page

A statically generated Page that pulls content directly from a Notion database. It uses `react-notion-x` for rendering and Next.js ISR (Incremental Static Regeneration) for high performance.

##  Features
- **Zero CMS cost:** Use Notion as your CMS.
- **Pretty URLs:** Automatic mapping of Notion slugs to `your-site.com/slug`.
- **Smart Caching:** Minimizes API calls to Notion during builds.
- **Syntax Highlighting:** Full code block support with PrismJS.
